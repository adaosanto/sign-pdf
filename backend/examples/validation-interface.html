Validador de Assinatura

Verifique a autenticidade de documentos assinados digitalmente

Digite a assinatura digital de 32 caracteres encontrada no PDF
Hash SHA-256 do documento para verificação adicional

Como usar o validador:

  1. Abra o PDF assinado digitalmente
  2. Localize a assinatura digital no final de cada página
  3. Copie os 32 caracteres da assinatura
  4. Cole no campo acima e clique em "Validar Assinatura"
  5. O sistema confirmará se o documento é autêntico
